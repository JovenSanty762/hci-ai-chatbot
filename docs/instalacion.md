# Instalación en Servidor Interno

## Requisitos del servidor
- Ubuntu Server 22.04
- Docker y Docker Compose instalados
- 16 GB RAM recomendados

## Instalación
1. Clonar repositorio
2. docker-compose up --build -d
3. descargar modelo llama3
4. verificar endpoints

## Comandos útiles
docker ps
docker logs hospital_backend
docker logs hospital_frontend
docker logs hospital_mysql
docker logs hospital_ollama
