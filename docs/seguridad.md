# Seguridad y Privacidad

## Medidas recomendadas
- Acceso solo desde LAN hospitalaria
- Firewall bloqueando accesos externos
- No almacenar conversaciones completas en logs
- Enmascaramiento de documentos
- Auditoría por sesión
- Políticas de retención de datos

## Cumplimiento
Se recomienda alinearse con normativas locales de protección de datos personales.
